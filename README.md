# 🍞 Levain — Sourdough Companion App

A clean, mobile sourdough app built with React. Everything a home baker needs in one place.

**[→ Open the app](https://levain.pages.dev)**

---

## What's inside

- **Bread calculator** — hydration, starter %, salt, and whole grain ratios with live scaling
- **Pizza & flatbread calculator** — dough per ball, oven type, fermentation tips
- **Starter tracker** — feeding ratios and timing guide
- **Recipes** — flatbread, loaf pan, freestanding, rye, pancakes, pizza
- **Baking guide** — flour types, fermentation, shaping, scoring
- **Troubleshooter** — diagnose and fix common baking problems

---

## Stack

- React 18 + Vite
- Single `.jsx` file — no external UI libraries
- Deployed on Cloudflare Pages

---

## Run locally

```bash
npm install
npm run dev
```

Heavily inspired by **[Hendrik's The Sourdough Framework](https://github.com/hendricius/the-sourdough-framework)** — a rigorous, science-based open-source guide to sourdough. If you want to go deep, start there: [the-bread-code.io](https://www.the-bread-code.io/)
